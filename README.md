# DMF Knowledge Base

## Digital Monetary Framework — Fully-Backed Digital Currency

Welcome to the official Knowledge Base (KB) for the **Digital Monetary Framework (DMF)**, a protocol on the Base blockchain that issues **dmfUSD** — a fully-backed, non-custodial digital token. Every dmfUSD in circulation is backed 1:1 by USDC held transparently in an immutable smart contract. No lending, no leverage, no admin keys.

This KB is the authoritative reference for understanding how DMF works, how to use it, and how to verify its integrity.
